This folder contains Matlab tools to manipulate the H3D (Humans in 3D) dataset.
The demo file is "demo.m". It initializes H3D and runs several demos on it.

H3D is described in the following paper:
Lubomir Bourdev and Jitendra Malik, Poselets: Body-Part Detectors Trained using 3D Pose Annotations, ICCV 2009

The web page of this project is: http://www.eecs.berkeley.edu/~lbourdev/poselets
